순번 기능 key 값 설명
1 음성/영상 원본 파일 삭제 원본 파일 자동삭제 설정
2 음성 기록 파일 삭제 음성 기록 자동삭제 설정
3 AI 요약 삭제 AI 요약 자동삭제 설정
